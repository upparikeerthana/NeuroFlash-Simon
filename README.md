
# NEUROFLASH-SIMON

## Overview

**NEUROFLASH-SIMON** is a web-based memory game inspired by the classic Simon game. The application challenges players to memorize and reproduce a progressively expanding sequence of visual cues. This project demonstrates fundamental front-end development concepts, including event handling, animations, and game logic implementation.

---

## Features

* Interactive memory-based gameplay
* Progressive level difficulty
* Real-time game feedback
* Clean and responsive user interface

---

## Technology Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## Installation

1. Clone or download the repository
2. Open the project directory
3. Launch `index.html` in a web browser

---

## Usage

Start the game, observe the sequence, and repeat it in the correct order to advance through levels. The game ends when an incorrect input is entered.
